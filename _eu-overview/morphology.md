# Morphology

The UD treebank follows the UD guidelines for part-of-speech tags and features. When using some language-specific features, the treebank adheres to the features described in <a href="u/ext-feat-index.html">language-specific features</a>, as in Person[abs]=1, Number[erg]=Plur or Polite[dat]=Inf.
