# Tokenization

**This document is a work-in-progress proposal. See http://universaldependencies.github.io/docs for current/accepted specification** 

Word tokenization of Turkish is similar to the other languages writting using Latin alphabet. However, syntactic analysis of Turkish requires sub-word units as syntactic tokens.
